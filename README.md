# create-dta

CLI для создания DTA проектов на базе Turborepo.

## Использование
```bash
npx @dta/create-app my-project
```

## Опции
```bash
npx @dta/create-app my-project \
  --base=with-tailwind \
  --template=rbac+feature-flags \
  --package-manager=pnpm
```

## Фичи

- `rbac` - Role-Based Access Control
- `feature-flags` - Feature Flags система
